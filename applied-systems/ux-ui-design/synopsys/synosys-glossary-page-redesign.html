Synopsys Glossary Page Template




I got a request to redesign the glossary page and eventually be a page template.

purpose of redesign



01.
To gather keyword information.

02.
To make sure all glossary pages are consistent.

03.
To convince our users to jump to the attractive content to them.

04.
To suggest additional content for users.

The previous page look

The pain point



01.
Bad contrast.

02.
Too much additional information in right rail.

03.
Prioritization problem.

04.
Page is too long to find topics.

Glossary page new Design

There are several design updates on the page

The updates on new glossary page



Add a skinny banner with the title and the author's name, the author's bio page link, and the published date

Reason: To separate the navigation with the main page content. The title is clearly shown at the top of the page. Instead of using an image banner, a patterned banner will not take attention away from the title and authors.

Remove the right rail

Reason: After checking the data, Right rail content has low traffic. It doesn't need to be on the important top of the page.

Add a table of contents

Reason: The page is long with different topics. It will be easy to users for find their favorite topic.

Create a style for "Definition"

Reason: Make a special style for better contrast.

Add "Continue Reading" at the bottom of the page for related reading

Reason: Try to keep the users staying on the website longer by proposing new related content.

Build a scroll up arrow on the right bottom of the page

Reason: The long content page does not just need a table of contents and a function to jump back to the top.
